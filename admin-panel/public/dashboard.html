Fase 1: Conexion Segura (HTTPS)

Configura tu dominio con Cloudflare para SSL gratis. No necesitas el terminal.

1

Comprar un Dominio

Necesitas un dominio propio (ej: tuempresa.com). Lo puedes comprar en cualquier registrador.

Registradores populares: Namecheap, GoDaddy, Porkbun. Precio: ~$10/ano
2

Crear Cuenta en Cloudflare (Gratis)

Cloudflare te da SSL/HTTPS gratis. Crea una cuenta en su sitio web.

Abrir Cloudflare ↗
El plan gratuito es suficiente. No necesitas pagar nada.
3

Agregar Dominio a Cloudflare

En Cloudflare, agrega tu dominio y cambia los nameservers en tu registrador.

3a En Cloudflare: click en "Add a site" y escribe tu dominio
3b Selecciona el plan "Free"
3c Cloudflare te da 2 nameservers. Ve a tu registrador y cambia los nameservers a los que Cloudflare te indica
Los nameservers se cambian en donde compraste tu dominio. Puede tardar hasta 24 horas, pero usualmente toma minutos.
4

Crear Registro DNS (A Record)

En Cloudflare DNS, crea un registro A que apunte tu dominio a la IP de tu servidor.

IP Publica de tu Servidor
Cargando...
4a En Cloudflare: DNS > Records > Add Record
4b Type: A | Name: @ (o tu subdominio) | Content: la IP de arriba
4c Proxy status: Proxied (nube naranja activada)
Copia la IP de arriba y pegala en el campo "Content" del registro A en Cloudflare.
5

Verificar HTTPS

Abre tu dominio en el navegador con https:// y verifica que carga correctamente.

⚠ Verificando...
Si el candado aparece en tu navegador, HTTPS esta funcionando. Si no, espera unos minutos y recarga.
🛠

Fase 2: Instalacion de OpenClaw

Ejecuta estos comandos desde el tab Terminal.

6

Instalar OpenClaw

Instala el CLI de OpenClaw globalmente en el servidor

sudo npm install -g openclaw@latest
7

Instalar Claude Code

Necesario como provider de IA para OpenClaw

sudo npm install -g @anthropic-ai/claude-code
8

Login en Claude

Autenticarse con Anthropic. Va a mostrar un link en la terminal, copialo y pegalo en tu navegador

claude login
Tip: El link que aparece en la terminal lo debes abrir en tu navegador para completar la autenticacion
9

Configurar OpenClaw

Wizard interactivo que configura autenticacion, gateway y canales (WhatsApp, Telegram, etc.)

openclaw onboard --install-daemon
Este wizard te guia paso a paso. Puedes conectar canales como WhatsApp, Telegram, Discord, Slack y mas
10

Verificar Gateway

Confirma que el gateway esta corriendo correctamente

openclaw gateway status
11

Abrir Dashboard

Ve al tab "OpenClaw Dashboard" para ver el panel completo de administracion

Haz click en el tab "OpenClaw Dashboard" arriba. Si el indicador esta verde, todo esta listo

Modelos y Providers Disponibles

OpenClaw soporta multiples modelos de IA. Puedes conectar varios al mismo tiempo.

Anthropic (Claude) Incluido

Claude Opus, Sonnet, Haiku. Se configura con claude login (paso 8).

anthropic/claude-sonnet-4-6
OpenAI (GPT) API Key

GPT-4o, GPT-5, o1. Necesitas una API key de OpenAI.

openai/gpt-4o
openclaw config set models.default openai/gpt-4o
Google (Gemini) API Key

Gemini 2.0 Flash, Gemini Pro. Necesitas API key de Google AI.

google/gemini-2.0-flash
openclaw config set models.default google/gemini-2.0-flash
Ollama (Local) Gratis / Local

Llama 3, Mistral, DeepSeek. Corre en tu servidor, sin API key.

ollama/llama3.1:70b
curl -fsSL https://ollama.com/install.sh | sh
OpenRouter API Key

Acceso a 100+ modelos con una sola API key. Claude, GPT, Llama, Mistral, etc.

openrouter/anthropic/claude-sonnet
Otros Providers

xAI (Grok), Mistral, DeepSeek, Groq, Cerebras, Hugging Face, LM Studio, GitHub Copilot.

Usa el wizard openclaw onboard para configurar cualquier provider
